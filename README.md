# Social Media Analysis Dashboard

## Project Overview
This project analyzes Spotify social media and platform data using Microsoft Excel. The dashboard provides insights into user engagement, playlist performance, artist popularity, and listening trends through interactive visualizations and KPI tracking.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design
- Data Visualization

## Key Performance Indicators (KPIs)
- Total Engagement
- Likes
- Comments
- Shares
- Engagement Rate
- Platform Performance

## Objectives
- Analyze audience engagement across social media platforms.
- Identify top-performing content categories.
- Track engagement trends over time.
- Generate actionable business insights.
